🔐 Base64 Encoder & Decoder (Python)
📌 Description

This is a simple Python program that allows users to encode and decode text using Base64 encoding.
It provides a menu-based interface where the user can choose the desired operation.

🚀 Features
Encode plain text into Base64 format
Decode Base64 string back to original text
Automatically fixes missing padding (=)
Handles invalid input safely using exception handling
